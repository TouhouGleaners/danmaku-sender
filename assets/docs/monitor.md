# 弹幕监视器使用指南

此功能用于“实时对账”，监视 B 站云端是否真的收到了您的弹幕。

### ⚙️ 关键参数设置
*   **检查间隔**: 建议 **60 秒**。这是工具轮询 B 站接口的频率，太快可能导致 IP 暂时被禁。
*   **时间容差**: 建议 **500 ms**。
    *   *解释*: 本地发送时间是 `10.5s`，云端因为服务器处理可能会变成 `10.6s`。容差允许这两个值被视为同一条弹幕。(基本不会出现这种情况，但的确存在可能性)

### 🚀 启动监视
1. 确保已在“发射器”页面配置好目标视频和本地文件。
2. 点击 **“开始监视”**。
3. 进度条会显示已匹配的弹幕数量。

**💡 提示**: 
- 由于 B 站服务器存在缓存，发送成功的弹幕可能需要 **几分钟** 甚至更久才能在监视器中刷出来，请耐心等待。
- 由于 B 站阿瓦隆系统及其他因素影响，监视功能可能不准确，仅供参考。
