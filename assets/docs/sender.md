# 弹幕发射器使用说明

### 🍪 SESSDATA / BILI_JCT 获取
这是发送弹幕的通行证，请务必准确填写。

1. 在浏览器（推荐 Chrome/Edge）中登录 Bilibili。
2. 访问任意 B站页面，按 **F12** 打开“开发者工具”。
3. 切换到顶部的 **应用程序 (Application)** 标签页。
4. 在左侧菜单中找到：**存储 (Storage)** -> **Cookies** -> `https://www.bilibili.com`。
5. 在右侧列表中找到 `SESSDATA` 和 `bili_jct`，复制它们的 **值 (Value)**，粘贴到本工具的“全局设置”中。

---

### 🎮 核心功能
#### 1. 视频设置
- **BV号**: 输入视频 URL 中以 `BV` 开头的字符串。
- **获取分P**: 点击按钮，工具会自动拉取视频标题和所有分 P。
- **选择分P**: 必须在下拉框中选中一个目标分 P（只有一P时自动选择该分P）。

#### 2. 弹幕文件
- 点击“选择文件”加载本地 XML。工具会自动解析并检查格式。

#### 3. 发送策略
- **随机延迟**: 建议设置为 **8.0 - 10.0 秒**。
- **自动风控**: 如果遇到 `36703` (频率过快) 错误，程序会自动暂停 10 秒。

**⚠️ 提示**: 任务进行中，按钮会变为红色的“紧急停止”，点击可立即中断发送队列。
