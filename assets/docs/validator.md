# 弹幕校验器功能详解

**为什么需要校验？**
B 站对弹幕内容有严格限制。此功能用于在发送前“清洗”本地文件，剔除会导致发送失败的坏数据。

### 🛠️ 使用步骤
1. **准备数据**: 先在“发射器”页面加载好 XML 文件并选择分 P（需要视频时长来校验时间戳）。
2. **开始验证**: 切换回本页面，点击 **“开始验证”**。
3. **查看结果**: 问题弹幕会显示在表格中。

### 📋 处理方式
您有三种方式处理问题弹幕：

*   **✏️ 手动编辑**: 双击“弹幕内容”列，或 **右键点击** 条目进行修改。
*   **🗑️ 删除条目**: 选中一行或多行，点击底部“删除选中条目”或右键删除。
*   **✨ 批量修复**: 点击顶部的“批量修复”按钮，支持一键去除换行符或截断过长内容。

**注意**: 处理完成后，**必须** 点击右下角的 **“应用所有修改”**，否则发送时依然是旧数据！

---

### 🔍 校验规则标准
| 错误类型 | 说明 |
| :--- | :--- |
| **内容过长** | 超过 100 个字符 |
| **非法字符** | 包含换行符 `\n` 或 `/n` |
| **时间超限** | 发送时间点超过了视频总时长 |
| **特殊符号** | 包含某些可能导致解析错误的特殊字符 |
